# Personal Portfolio

A modern, responsive portfolio built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## 🚀 Recent Updates

- **Tools App**: Added direct link to the Tools ecosystem in the Hero section.
- **Enhanced Focus Areas**: Updated professional highlights to emphasize Leadership, Automation Mindset, and Agentic AI expertise.
- **Linting & Optimization**: Cleaned up code for better performance and error-free builds.

## 🛠️ Features

- **Hero Section**: Impactful introduction with quick action buttons.
- **About Section**: Detailed story and professional highlights.
- **Projects**: Filterable project gallery with GitHub and Live demo links.
- **Skills**: Categorized technical expertise.
- **Experience**: Career timeline from Programmer Analyst Trainee to Support Team Lead.
- **Contact**: Quick links for Email, LinkedIn, and Phone.
- **Theming**: Dark/Light mode support.

## 💻 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```
---------->
npm run deploy


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🏗️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev)
- **Deployment**: GitHub Pages / Vercel
